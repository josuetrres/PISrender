Diseño de Circuitos para Nuestro Robot Recolector

1. Servomotores

Los servomotores son fundamentales para el movimiento preciso de cada articulación de nuestro brazo robótico. Implementamos señales PWM (Modulación por Ancho de Pulso) para controlar el ángulo de rotación de cada servo. Esto nos permite un control fino sobre el movimiento del brazo, esencial para la recolección eficiente de basura.

Utilizamos servomotores de alto torque para las articulaciones principales y servos más pequeños para los movimientos finos de la pinza. La implementación de un controlador PID nos ayuda a mantener la precisión y estabilidad en el movimiento.

2. Sensores

Implementamos varios tipos de sensores para optimizar el funcionamiento de nuestro robot:

  • Sensor de temperatura: Monitorea constantemente el ambiente para asegurar condiciones óptimas de operación y seguridad para los humanos cercanos.
  • Sensor de fuerza y peso: Instalado en la pinza del brazo, nos permite medir el peso de los objetos recogidos y ajustar la fuerza de agarre para evitar daños.
  • Sensores de proximidad: Ubicados alrededor del carrito, estos sensores ultrasónicos detectan obstáculos y ayudan a navegar hacia los puntos de recolección de basura.

Todos estos sensores están conectados a nuestro controlador principal, permitiendo una toma de decisiones en tiempo real basada en los datos recopilados.

3. Módulo GPS

El módulo GPS es crucial para la navegación autónoma de nuestro carrito. Utilizamos un módulo GPS de alta precisión que nos proporciona datos de latitud, longitud y altitud. Esto nos permite:

  • Rastrear la ubicación exacta del carrito en tiempo real.
  • Planificar rutas eficientes para la recolección de basura.
  • Crear un mapa de las áreas limpiadas y por limpiar.

Integramos los datos del GPS con un algoritmo de planificación de ruta para optimizar el recorrido del robot y maximizar la eficiencia de recolección.

4. Motorreductor y Controlador de Motor

Para el movimiento del carrito, utilizamos motorreductores de alta eficiencia. Estos motores proporcionan el torque necesario para mover el carrito incluso cuando está cargado de basura. El controlador de motor nos permite:

  • Controlar la velocidad y dirección de cada rueda independientemente.
  • Implementar giros precisos invirtiendo la polaridad de los motores.
  • Realizar un arranque y frenado suave para evitar sacudidas en el brazo robótico.

Utilizamos un controlador de puente H para manejar los motorreductores, lo que nos permite un control bidireccional y eficiente del movimiento.

5. Controlador ESP32

El controlador ESP32 es el cerebro de nuestro robot. Este potente microcontrolador se encarga de:

  • Controlar los servomotores del brazo robótico.
  • Procesar los datos de todos los sensores implementados.
  • Manejar la comunicación con el controlador de motorreductores.
  • Enviar y recibir datos a través de Wi-Fi para el monitoreo remoto.

Aprovechamos la capacidad de procesamiento del ESP32 para implementar algoritmos de control avanzados y toma de decisiones en tiempo real, mejorando la autonomía y eficiencia de nuestro robot recolector.

6. Controlador ESP32-CAM

El ESP32-CAM es una adición crucial para la funcionalidad visual de nuestro robot. Este módulo:

  • Crea un servidor web que transmite video en tiempo real.
  • Está montado en la parte delantera del carrito para una visión clara del entorno.
  • Permite a los operadores remotos ver el área de trabajo del robot.
  • Facilita la identificación visual de basura y obstáculos.

Integramos la transmisión de video en nuestra interfaz web principal, permitiendo un control y monitoreo completo del robot desde cualquier dispositivo con acceso a internet.
